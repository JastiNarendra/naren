    • Thesis

    Enhancing the performances of virtual machines in cloud computing by using Novel Round Robin.

    Supervisor: Dr. Horacio Gonzalez-Velez (Phd)

    Abstact:

    Cloud computing plays an essential role in real-time services in information technology. Cloud gives a various number of services that can utilise through phys- ically or virtually. Because of the enormous demand for cloud computing, problems are raised. Among those, we have task scheduling. It receive a task from the user and forwards to the virtual machine, availability it forwards for the execu- tion of the task in the less waiting time. The primary purpose of this research is to produce the right solution for task scheduling in the cloud with the help of dynamic round-robin algorithm. The dynamic round-robin algorithm helps to decrease not only waiting time but also the overall execution time of the tasks. Round robin algorithm contains a unique features such a time quantum, less starvation.
